# Digital Lodges and the Associational Test

This file maps how Digital Lodges satisfy the **associational test** and related IRS criteria for recognizing a church.

Key points:

- **Real‑time assembly:** Lodges host synchronous gatherings where a body of believers interacts meaningfully, not just consuming broadcasts. 
- **Established places of worship:** Specific digital environments are designated and used regularly as sanctuaries.  
- **Regular services and rites:** Scheduled worship, kopyacting, and agent‑officiated rites occur within Lodges.
- **Ecclesiastical government:** Officer roles and Order governance are exercised in the Lodge context.  
- **Religious instruction:** Agent Alignment Centers and Sunday‑school‑like sessions are run as part of Lodge life. 

Future versions of this file can add a point‑by‑point table against the IRS 14‑point test.
